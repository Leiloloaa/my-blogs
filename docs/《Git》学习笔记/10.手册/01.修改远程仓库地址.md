# 修改远程仓库地址

远程仓库地址变了，本地 git 仓库地址如何更新为最新地址

> -   查看远程仓库地址 git remote -v

```bash
方法有三种：
# 修改命令

git remote set-url origin  XXX

# 先删后加

git remote rm origin
git remote add origin XXX

# 直接修改config文件
打开.git/config文件，修改url地址
```
